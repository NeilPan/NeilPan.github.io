+ 武陵。風景 +

一些隨拍的風景稍晚上菜囉 ^^

=============================

開了好久，也睡了好久，我們終於要進入武陵囉 ^^

人家說行車時頭手不要伸出車外，我是都伸出去了

進入武陵沒多遠，我們就被風景吸引，準備下車走走，我一開車門就看見今天第一抹紅 ^^

其實我自己並不是很清楚，第一個點地名叫做什麼 @@

大概是因為在國內的關係吧，國內的旅遊我大概都不會太在意地點名

反正比較容易到達 ^^ 國外的話我就會很注意了，因為事前也會做很多的規劃

到達這個不知名的地點已經是中午，頂光卻還是一片藍，這就是海拔高加上空氣乾淨的關係囉

由於最近拍人像可能比較希望有光影或者顏色的對比，所以不知不覺連拍風景也被影響了

(當然除了shine兩口子那張例外，那張叫做搞笑側拍照，搞個枯樹當背景，是要後製成哈力波特嘛 XDDD )

接下來我們到了渡假村附近，先停了車，走路到溪邊櫻花勾吻鮭的保育地

光線已漸西斜，穿過樹林縫隙間的陽光，讓舉目所見更有層次，就連溪水也清澈見底

一伙人在這裡隨意的抓拍人像，風景，最後還拍起鳥來

我還是繼續我的光影與顏色的遊戲 ^^

等到太陽終於被山擋住後，我們一行人回到渡假村 check in

武陵富野渡假村真的是怕遊客無聊，住宿安排DIY竹筒飯的行程

還在走廊間設置了一堆益智遊戲

米米運氣很好的終於拼成一個正方體，而我只是隨便亂拼成一個好像尼斯湖水怪的東西

晚餐後原本想出來看看銀河，卻因為月亮太亮了，導致星星反而黯淡.....

第二天天氣依舊晴朗無雲，離開渡假村時，溫度計還顯示24度哩

我們緩慢的出發前往福壽山農場，可惜福壽山農場的楓葉狀況沒有武陵的好，遊客也很多

我們在農場旁的小吃攤隨便解決了午餐，路邊的野貓一直對我們凹凹叫 @@

離開福壽山農場後，我們繼續往高處走，前往算是本次旅行最後一站，天池

天池的光線，也是這趟旅行我覺得最美的

錯落在樹梢間，完完全全的光影，既柔和溫暖，又有足夠的強度

這種光線，怎麼拍都美，不拍也很美囉 XDDD

離開這裡後，我們就慢慢的下山，結束這趟難得的旅行

臨走前築楓的最後一拍，正好陽光也灑了下來

此時我除了覺得山在歡送我們以外，更覺得

【怎麼這時會是築楓站在那裡......XDDDDD】

==========================

感謝築楓安排行程，兩位車長辛苦的駕駛

其實我們後來還有再往更高一點去，景色更純粹，只是光線已經不夠美了

這也是明年想要開三天兩夜，武陵一路殺到埔里團的緣故

期待明年11月底12月初囉 ^^y

感謝賞圖 <(.)>

+ 清境。人像。風景 +

清境，有羊 ^^

============================

自從去年退伍後，上來清境住了幾天，便下定決心每年都要上來一次

乾淨而且冰涼的空氣，一直是我的最愛

這裡的民宿大多充滿歐式風格，更讓整個清境像童話世界一樣

如果有天真的海平面上升，我們現在住的平地都被淹沒的話

我想我會馬上逃到清境吧

去年上來時，都是超美的藍天，今年卻見到了完全不一樣的清境

飄小雨，不知道是山嵐還是霧，一直盤据在山頭，日夜不散

又是另外一種感覺，期待明年的清境會給我什麼驚喜囉 ^^

今年上清境前，順道繞到奧萬大去，那時天氣還很好

不過時機點太早了，所以當然沒有楓葉囉.......

離開奧萬大，到達清境已經快下午兩點，吃飯前還有藍天

沒想到吃完飯後，整個清境已經被霧籠罩

不過遊客觀賞剪羊毛秀的興致還是不減 XDDD

每次陽光露臉的時間大概只有幾秒鐘，所以也就沒有拍什麼了.........

這次住的地點是，香格里拉空中花園

去年發現他有調酒吧，但去年因為正在整修，所以沒喝到

想說今年一定要喝到，所以上來前就二話不說定了這家

結果！！！今年調酒師因為去旅展幫忙，不在山上......Orz

整日的霧，讓我們想看銀河也沒機會了

只能記錄下這個灰濛濛的清境夜晚，外面的溫度大概是12度

民宿裡住滿了人，來來往往，好像一個家庭

晚上跟米米一起窩在房間看HBO，這就是我要的悠閒吧 ^^

【旅行。趣】七美。全輯發表

6/24更新

===================================================

從開始報名到出發，中間經過三個多月(吧？)的七美望安澎湖行，終於在上週末出發

北海道回來後，接著期盼的就是這次活動，隨著出發時間的接近，柯柯也終於在Blog宣布「大家七美見哩.....」

19號一大早，所有人聚集在松山機場，準備分成兩班飛機前往馬公，再一起搭船前往七美

雖然一路上天氣都陰陰的，甚至在馬公的遊覽車上與往七美的船上甚至下雨

不過到了七美後，下午的狂爆藍天與真的會咬人的陽光，讓所有人怎麼想也想不到我們將受困七美四天XDDD

其實，這是我第一次參加這麼大團的外島團，以前總耳聞外島是多麼美，每年出團必有求婚大作戰

沒想到，七美固然真的很美，純樸充滿台灣味的居民，一望無際的海天一色，隨隨便便就出景的夕陽

讓人目不暇給，讓我們在第一天還風平浪靜時，不顧炎熱騎著機車試圖遊遍島上的每一個角落

除此之外，居然本團也有求婚大作戰，這好像真的是外島團的傳統，每年都會有情侶求婚 @@

在七美的第二晚，所有人暫時忘記了因颱風無法繼續行程的遺憾，真心話大冒險玩到民宿屋頂快掀起來

並且所有人偷偷的配合原訂計畫，讓元宏的溫馨又很拼體力的求婚計畫順利完成 XDDDDDD

持續滯留的第三天，雖然風雨交加，但當民宿老闆娘送來熱騰騰的海鮮粥時，所有人都齊聲歡呼(真的快要變難民)

最後，終於在Markur與柯柯的大力奔走下，我們可以趕在週一晚上回到台北

第一次的外島團，卻好像什麼都有了，有美麗的風景，極盡搞笑的大師群，有求婚，有颱風

四天沒有電腦、網路、msn、email的日子，卻是難得的經驗

感謝柯柯、Markur辛苦的主辦，偶而放逐，遠離原本的生活，也是一種生活的方式囉

北海道還有兩天沒po，看來圖要欠很大了 Orz，先來個文字記錄，免得到時想寫也寫不出來

期待眾位大師的作品，小弟慢慢龜圖去 XDDDDDDDD
