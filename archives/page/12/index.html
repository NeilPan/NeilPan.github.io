我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day2

經過一夜的休息，到香港的第二天，我們準備前往澳門

港澳常常是綁在一起的行成，因為進入澳門不需要簽證

只要在港澳碼頭買張船票，大概坐一個多小時的噴射遊艇，很快就到達澳門

需要注意的是，因為從香港到澳門屬於出境，從澳門再回香港又等於再入境香港

所以港簽上面需要有四格才能自由進出喔，不過通常申請的港簽就會是四格啦

從台北進香港 => 香港至澳門 => 澳門回香港 => 香港回台北，正好蓋滿四格 ^^

想要從香港去澳門的朋友們，不需要太擔心，很方便的

廣仲的早安晨之美如果要改成港版，那大概就是早安茶餐廳了吧 ^^

離開酒店後，我們在威靈頓街上的一家茶餐廳解決早餐

也許是觀光客沒這麼早起，也許是其他觀光客住的地方都有早餐

一進店內，其實沒有什麼觀光客的蹤影，幾乎都是本地人

這也讓我們更覺得貼近香港的生活，畢竟在這裡，早餐吃個茶餐廳也是一件很合理的事情 XDDD

我們點了波羅油，牛奶麥皮，跟三文治，再加上每餐必出現的奶茶，其實份量還蠻足夠的

店裡是一片昏黃，結帳處上面的電視播放奇奇怪怪的談話節目

在台北，要早餐可以吃茶餐廳似乎是件不簡單的事，因為好像幾間有名的茶餐廳都11點才開門

一大早的茶餐廳，也是香港的特色吧

吃完早餐後，我們一路步行前往港澳碼頭，昨天太陽還有露臉，到今天已經是霧濛濛一片

香港連綿的天橋又是另一個台灣少見的特色，我所能想到類似的，也只有台北信義區

在港澳碼頭買了票，我們繼續往海關的方向前進，卻沒想到碰上大批人潮 (海關不能照相)

看起來大家都會順便去一趟澳門的樣子

而且不曉得是人真的太多還是假日海關人員比較懶，出境的隊伍行進的頗慢

害我們差點趕不上船 @@|||

噴射遊艇駛離香港後，就進入一片白濛濛的海，所以我就在船上昏死了 XDDD

一直到澳門後，我們在碼頭附件轉搭威尼斯人酒店的接駁車，直接往這個頗負盛名的澳門渡假村前進

在威尼斯人，更正，應該說是在整個澳門，我都沒有拍很多照片

我們到達威尼斯人後，一路穿過他們的賭場，中間當然也見識了富麗堂皇的內部裝潢

賭場好像是禁止攝影的，所以我也不敢拍，免得旁邊戴扁帽看起來像情報員的人來關心我

至於威尼斯人大廳的屋頂，我也只是簡單的壓幾張，畢竟七月就要去義大利，威尼斯人這種程度的我根本懶得看.....

值得一提的是FF+17mm居然屋頂還照不進來，如果有朋友想來這裡，建議帶個魚眼會更不錯 XDDD

我想，如果要拍攝人像，澳門是可以去的，因為他最大的好處就是離台灣近

又有一些台灣少見的建築，顏色也很鮮艷特別，要有異國風也不是難事

可是如果想要旅遊，想要認識一些新的東西，澳門實在是無聊透頂，因為他實在是過度人工化

我們從威尼斯人，坐上的士前往澳門的婚紗攝影熱門點 -- 龍環葡韻住宅式博物館

為什麼說婚紗熱門景點呢，因為我們在這邊遇到一大群來拍婚紗的新人

感覺應該只有中間有白頭紗的是新娘，不過他們伴娘也太多了吧，還是大家都出來拍一拍玩一玩 XDDD

也讓我順便拍到這群背影，嗯嗯我看看...有LV，有COACH，好像還有GUCCI，哈

至於這個博物館，對我來說就像是漆不同顏色的淡水小白宮

當天還有鬱金香展，不過也沒我們的桃源仙谷大片，總而言之，只有這群有趣婚紗攝影吸引我吧

長焦段在澳門也是蠻好用的，這張是180/2.8D拍的

接下來，我們繼續坐的士往下一個地點 -- 西望洋聖堂，以下取自維基：

西望洋聖堂，又稱為主教山聖堂，海崖聖母小堂，卑拿聖堂，西望洋聖母堂，位於澳門西望洋山，建於西元1622年

卑拿在葡萄牙文義為位於海邊之山岩，聖堂多次修整，1935重建為現在的規模

聖堂旁邊的建築物曾經是主教的寓所，所以又稱為主教山小堂

建築物蓋有歌衛式教堂圓頂，反映古典折衷主義風格，附件的路德聖母岩洞設立聖母像

象徵聖母曾在法國路德城的顯現事件

這個小教堂似乎罕有人至，或者說一般觀光客不太會來這裡，導致我們還跟運將講很久

還拿地圖直接指給他看，他才知道教堂在哪裡

到了教堂，天空已經開始飄起小雨，教堂裡面疏疏落落的幾個人正在祈禱

聖母像則在一片陰霾的天空中望著遠方的海洋.......

離開西望洋聖母堂後，我們要往澳門最後一個地點，著名的大三巴牌坊週邊

不過我們錯算了一件事，就是根本不會有的士上來西望洋聖母堂，所以我們只好一路走下山

而且主教山算是澳門比較偏僻的地方，居然連下山後也看不到一輛的士

我們一路往大三巴牌坊走，等到終於出現的士了，其實我們也快到了 = =.........

大三巴牌坊附件又屬於觀光重點區，人潮洶湧，我們彷彿從鄉村回到都市

這時我因為已經背相機背到累了，所以最後索性收DDD，只用GF1隨便拍

這邊的律師樓都直接叫某某某大律師，也很妙 XDDD

我們在牌坊週邊繞了一圈，吃了難吃的豬扒堡，便直接回碼頭坐船回香港

回到酒店附近，餓到已經受不了的我們，準備在酒店樓下的蓮香樓大快朵頤一下 ^^

蓮香樓也是香港有名的餐廳，正巧就在蘭桂坊那條路的路口

一進門，這家餐廳的服務生，全部都是至少40歲以上年紀的熟男

只有櫃台結帳處是一個阿婆 XDDD

菜色的名字我一樣也不記得了，只知道這應該是我在香港吃的最開心最撐的一頓

由於我跟米米都盡量吃素的關係，除了叉燒肉是我刻意點的以外，其他全是不含肉的菜色

那為什麼會有叉燒出現呢？其實只是我想在香港一嘗道地的叉燒，畢竟實在是被食神影響太大了 XDDD

米米也吃了一小塊，其餘的叉燒全都是我吃光，的確很好吃，肉嫩皮脆，又香又有油脂

不過我不否認一個人吃光一盤到最後是有點噁心 XDDD

下次還有機會來香港的話，蓮香樓我一定會再回來 = =+

晚餐過後，我們就回到房間休息了，明天是最後一天的血拼日^^

我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day1 晚上

從赤柱回到市中心後，我們在酒店房間稍做休息，並且拿起腳架準備上太平山看夜景

從中環坐地鐵到上環，再一路沿著指標散步到山頂纜車站

原本的計畫是輕輕鬆鬆的坐纜車上山，看個夜景，或許還可以參觀一下杜莎夫人蠟像館

結果一到纜車站就被人龍嚇到 @@|||

人潮沿著纜車售票口，一路像人體內的小腸一樣排隊，在車站外圍人行道上繞了好幾圈，都是要上山的人

紅色的的士一台台停在路邊招呼排不下去的觀光客，可是我們只有兩個人太不划算，只能乖乖的排隊

我想我們大概排了至少一個小時有吧 = =|||

而且有坐過的朋友就知道，太平山的纜車只有一台，載上去再載下來，導致隊伍真的以很慢的速度推進

還好這是我們今天最後一個行程了

在排隊時間，我才知道原來太平山的纜車不是像貓纜那樣，而是有點像我們阿里山的小火車

是走在鐵軌上的，而且因為幾乎是直線上去，所以坡度很陡

我們沒有搶到位子，所以斜斜的站著，一路看著夜景上山，也是一種新奇的體驗 ^^

這就是香港百萬夜景，嗯，老實說我覺得還好，可能我本來就對夜景沒啥興趣吧 XDDD

即使時間已經有點晚，山頂風也挺冷的，凌霄閣的遊客還是非常多，山頂上還有幫遊客拍到此一遊照的服務

台北市的夜景我常常看，甚至我還看過花蓮的夜景

台北市的夜景最突出的一定是101了，而花蓮的夜景則是田埂間的路燈刻劃出一片片稻田的輪廓

而香港，密集的高樓林立，又是另外一種特別的感覺

離開凌霄閣後，我們在廣場附近的一間餐廳外，據說上太平山一定要來拍這個"太平山" XDDD

不過餐廳賣的東西一樣很貴，所以我們只吃了麥噹噹，變晃到杜莎夫人蠟像館

一樣不免俗的跟蠟像拍一些搞笑的照片 XDDD

其實，我是要跟李小龍跳slow探戈的，不過我想沒有人猜得出來吧

下太平山的纜車一樣又是一條人龍，我們直接放棄搭纜車下山的計畫，轉搭巴士

巴士站的人，相較於纜車實在是少的多，大概等不到20分鐘我們就上車了，回到酒店房間準備慶祝一下 ^^

老實說，2010年1月1日是我28歲的生日 XDDD

很巧，巧到我懷疑蘭桂坊有特地安排的是，我們的房間正好在28樓，1號房，所以房間號碼是2801 @@

樓層的的標示是用燈投射的方式，房卡也有專用的卡套，記載著房號及入住與退房時間

卡套我當然把他帶回台灣了，蛋糕則是在地鐵站的蛋糕店買的

兩隻老鼠(一隻被擋住)圍著一塊起司造型的蛋糕，老鼠本身是巧克力，老實說不太好吃

蛋糕部份到還ok，後來我們退房時就把蛋糕留在房間冰箱裡 XDDD

我的28歲，有米米跟香港的萬家燈火陪我度過 ^^

最後在放一張房間看出去的夜景，明天，我們將搭船前往澳門 ^^

我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day1 下午

在翠華結束到香港的第一餐後，我們出發前往香港島的南方 -- 赤柱

在規劃香港行程之初，我們考慮在赤柱與淺水灣間選一個

後來選擇了赤柱，是因為淺水灣似乎比較多人去過，而且可能只有海灘

而赤柱除了也是海邊外，還有赤柱市場，香港的市集也是我們此行的重點

搭雙層巴士到赤柱其實蠻快的，車程大概也才花三四十分鐘

從上環一路經過香港的商業中心金鍾，在一路往南邊經過香港的高級別墅區

景色已從夾道林立的摩天大廈，人潮洶湧的大馬路

逐漸轉向可能只有兩三層的別墅，窄窄的僅容單線雙向的山路，以及路旁成排的樹蔭

變化很大，卻還是香港

好像是啥中國人民解放軍大廈

迅速通過高等法院，還好DDD夠快 XDDD

第三天我們會搭這個去時代廣場 ^^

香港其實真的很多.....看起來像外勞的外國朋友，不過我真的很懷疑他們是不是外勞，還是是觀光客

注意看左下角那位還拿LV包哩，在台灣我可從來沒看過外勞朋友這麼高調

右邊那棟是香港賽馬會，其實本來此行有要考慮進賽馬場看看，畢竟那也是特色之一

不過後來沒去，沒時間 T__T

穿過隧道，我們已經慢慢離開都會區，沿著香港島南邊的海岸線，一路抵達赤柱

雙層巴士沿著海岸線行駛，我原本以為台灣的公車司機算技術非常好了

沒想到香港的雙層巴士司機真是讓我五體投地的佩服 @@

一路往赤柱的路上我們經過香港的高級別墅區，別墅區的路都很窄

而且沿山坡建築，一路高低起伏，彎彎曲曲，可是我們的雙層巴士司機居然像飆車一樣

我跟米米運氣很好，坐到上層的車頭位置，好幾次眼看車頭好像快撞到人行道的護欄，卻總是差幾公分

至於兩旁長得太茂盛的小樹枝就比較倒楣，司機一點也不在乎的就衝過去 XDDD

甚至司機還在一個狹窄的T字路口瞬間靠左，然後一個360度大迴轉再往原路回去

也是一樣，眼看迴轉時快要撞到路邊的標示桿，卻又硬是擦過去

而且重點是司機好像老神在在，知道絕對不會撞到，幾乎沒特別再踩煞車 @@

就這樣經過特技表演，我們在赤柱廣場下車

在一路步行經過兩旁都是餐館的赤柱大街 (東西非常貴，我們都吃不起 Q___Q)

往赤柱市集前進

走過赤柱大街，我們到了市集的入口處，米米馬上在這邊解決紀念品的問題，我則是東看西看，順便偷拍

應該是因為假日的關係吧，整個大街與市集人都非常多，而且似乎各國人都有 @@

接下來我們進到市場更內部，整條街更是人擠人，有點像假日的九份老街

不過九份早就太商業化了，無聊的很，赤柱市場還是有很多很有特色的店 ^^

赤柱市場大概最多的，除了傳統服飾或布料的店外，便是畫攤了

各式各樣的畫，香港的夜景，街景，都在這裡

有趣的扛棒又來了 XDDD

在香港另一個我很喜歡看的，就是他們的電影海報，特別是翻譯

裘德洛 => 祖狄羅，鋼鐵人 => 鐵甲奇俠，即使翻譯也仍有濃濃的港味 XDDD

接下來這個西醫更囂張，內外全科醫學士，不仔細想想還以為他都能看哩，不過我想應該是指我們的家醫科吧

家醫科的確是什麼都能看，可是要更專業點還是要個別科呀 XDDD

就這樣，我們在赤柱的巷弄裡穿梭尋找新鮮事，度過到香港第一天的下午

然後一樣坐上雙層巴士，回到上環，晚上我們即將一覽百萬夜景 ^^

我們在魔幻之都尋找似曾相識的風景 -- 2010元旦港澳之旅 Day1 上午

終於在漫長又發霉的年假中，把香港的圖修出來 Orz

(年底真的很忙嘛)

===========================================

北海道一行回來後，米米和我就在計畫下一個地方要去哪裡

畢竟自助旅行是會上癮的 XDDD

好不容易元旦有三天長假，我們便決定到香港澳門過新年

港澳幾乎就像台北飛恆春一樣，沒花多少時間，飛機就已經落地

所以連睡覺都省了，下了飛機也也沒有什麼特殊的感覺

一直到靠近海關，我才因為港澳對於事物的特殊名稱，終於感覺到我踏進了香港

一個我闊別十年不只，印象中上次來是住在一個小小公寓，有著窄窄的走廊的地方

海關大堂

其實我覺得很多我們倒背如流的港片，例如星爺系列啦，無間道系列啦，還有其他很多系列

第四台都應該播放香港話的版本，因為他們的語言，有時總是更直接且貼切的描述一樣事物

海關大堂，確實挺大，而且大堂好像有點包青天要"升堂~"的感覺 XDDD

接機大堂 XDDD

聖誕節的裝飾還在，據說香港是一個非常有過節氣氛的地方，包括聖誕節，元旦，很多人都選擇來香港跨年

我沒有這個幸運(也沒這個錢跟假)，只選擇在元旦早上踏進這個地方

至於在香港幾乎每餐必喝的奶茶，老實說不是很對我的口味，但不難喝也很有特色

我很好奇香港人喝他們自己的奶茶到底加不加糖

在翠華酒足飯飽以後，我們按照行程，前往公車總站，搭車前往香港島的南邊 -- 赤柱

穿梭在大樓間，偶而露臉的陽光反射在大樓玻璃上，似乎宣告著下午天氣會更好

我們正式開始港澳之旅 ^^

=============================

開了好久，也睡了好久，我們終於要進入武陵囉 ^^

人家說行車時頭手不要伸出車外，我是都伸出去了

進入武陵沒多遠，我們就被風景吸引，準備下車走走，我一開車門就看見今天第一抹紅 ^^

其實我自己並不是很清楚，第一個點地名叫做什麼 @@

大概是因為在國內的關係吧，國內的旅遊我大概都不會太在意地點名

反正比較容易到達 ^^ 國外的話我就會很注意了，因為事前也會做很多的規劃

到達這個不知名的地點已經是中午，頂光卻還是一片藍，這就是海拔高加上空氣乾淨的關係囉

由於最近拍人像可能比較希望有光影或者顏色的對比，所以不知不覺連拍風景也被影響了

(當然除了shine兩口子那張例外，那張叫做搞笑側拍照，搞個枯樹當背景，是要後製成哈力波特嘛 XDDD )

接下來我們到了渡假村附近，先停了車，走路到溪邊櫻花勾吻鮭的保育地

一伙人在這裡隨意的抓拍人像，風景，最後還拍起鳥來

我還是繼續我的光影與顏色的遊戲 ^^

等到太陽終於被山擋住後，我們一行人回到渡假村 check in

武陵富野渡假村真的是怕遊客無聊，住宿安排DIY竹筒飯的行程

還在走廊間設置了一堆益智遊戲
