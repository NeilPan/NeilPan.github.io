【旅行。趣】初夏的北海道 DAY-3

先來張小樽必拍芭樂照 XDDDDDDD

=================================

來到北海道的第三天，我們的行程是小樽，岩井俊二【情書】拍攝的地點

因為昨天的美瑛行實在累壞了，所以我們放棄了原本搭一早的JR到小樽的計畫

決定先在飯店吃個早餐，不要再那麼趕

所以我們到了小樽，差不多已經是上午九點多囉

上午的行程是手宮公園，情書中男藤井樹騎著腳踏車經過一個下坡

將牛皮紙袋套在同樣騎著腳踏車的女藤井樹的頭上

傳說中的那個下坡，就在手宮公園，我們預計搭小樽巴士前往

JR票當然是不能用於小樽巴士的，所以我們每人付了240羊，一路前往手宮公園

其實我們並沒有找到手宮公園的門口，手宮公園位於山丘上，我們是沿著山腳，隨便找一條路上去

沿路經過半山腰的住家，還經過小小的神社，太陽依舊猛烈，我們邊走邊找往手宮公園的路

經過一連串上坡，我們真的進入手宮公園的範圍，可是，情書拍攝的季節是冬季

我們是初夏前往，再加上對手宮公園又不是那麼熟，要找到那個傳說中的下坡，當然是難上加難

所以，我們挑選一條下坡旁的一片草地，把這裡當作是男藤井樹當時的起點，向下眺望

「お元气ですか〜？」「私は元气です〜！」  電影裡博子是在天狗山的滑雪場喊著這兩句話，向樹道別

我們因為行程的關係，所以在這片草地上喊著

離開手宮公園的途中，意外發現這一長片落葉堆，落葉堆非常深且柔軟

人一踩下去，會一直陷到腳踝都不見了呢 :)

離開手宮公園後，我們步行前往下一個情書拍攝景點 -- 舊日本郵船小樽支店

路途中看到這台可愛的巴士，潔思米立刻就吵著要合拍拍～～～

仔細一看車長還坐在裡面呢！還好他沒突然把車開走 Orz

跟可愛的巴士合照後，我們進入舊日本郵船小樽支店，這裡是傳說中長大的女藤井樹工作的圖書館的景

一進去便是古色古香，天花板垂吊下來的一盞盞燈，有點像星星

這樣感覺的地方，應該是我們此行第一次遇到，這也顯示我們更接近有很多古老建築的小樽

整個支店並不大，只有一二兩層樓，可是走進館內沈靜的感覺，又跟在外面陽光普照的感覺大不相同

館內的服務員老伯伯熱心的跑來幫我們介紹，不過他好像只會說日文的樣子

我們是整個有聽沒有懂，後來，老伯伯拿給我們繁體中文的介紹

我們才知道，館內一定要看得是這片金碧輝煌的壁紙，由於年代久遠，四周的壁紙很多都已脫落

只有這片還算完整，搭配著古老的宮廷椅，有種繁華卻又已退盡繁華的感覺

但是，就這樣晃了一圈，情書中圖書館的場景，究竟在哪裡呢？

明顯的館內根本沒有書架這種東西，正當我們納悶，並且回到門口準備離開時

才發現，門口售票的櫃台好眼熟，原來，這裡就是長大的女藤井樹在回信給博子

女藤井樹的同事問，妳到底要不要來幫忙啦？的那個窗口

想必當時中山美穗就是坐在這個小房間裡，回著信吧

離開舊日本郵船小樽支店後，我們再搭乘巴士回到小樽市中心

肚子已經餓得咕嚕咕嚕叫，當然是前往小樽有名的壽司通，吃我們除了拉麵以外的午餐

本來想要去小樽有名的政壽司，也已經走到門口，不過實在因為政壽司太貴而作罷

後來我們索性往運河方向走，依照旅行書上面的介紹，找到這家握なかSUSHI

(其實也沒比政壽司便宜，這邊的壽司店價位都差不多......Orz)

那.....怎麼沒有壽司的照片呢？因為當時被價錢震撼到，加上又很餓，所以壽司一來就忘記拍，直接吃光了XDD

整體來說其實他們的握壽司還好，台北也吃的到這種程度的，算是有點失望 = =

小樽堺町這條路上，聚集了網路上來小樽必推的景點與名產店 XDDDDDDD

北一硝子、音樂盒堂、浪漫館、銀之鐘、北菓樓、LeTAO、雪印，還有好多古老的建築都在這條路上

我們沿著這條路走，有興趣的店就進去逛逛，路上常常可以看見馬車與人拉車

一條小小兩線道的馬路，聚集了所有來到小樽的觀光客 : )

我們邊走，邊吃超級好吃的冰淇淋 (好像吃了三支XDDDDDD)

其實堺町沿路的商家，大部分都因為做觀光客的生意，進去便是玻璃或手工藝品

價錢都不便宜，店家內其實也沒什麼好拍的，唯一我們有興趣的便是北一硝子三號館

在網路上看別人拍的照片，館內是那麼美，卻也查到似乎不能在館內照相(只能偷拍，拿D3怎麼偷拍 = =||)

還好，當我們真的到了三號館，才發現原來是禁止用閃光燈而已，不是不能拍

此時就是高ISO超強的D3的天下啦～～～～～～～～～～～～～

另外，其實三號館是一個咖啡廳，也賣些糕點、冰品，他們的紅豆湯圓牛奶冰超好吃的啦 ^^b

離開北一硝子後，我們繼續往Meruhan十字路前進，途中經過我這次小樽行必至的北菓樓

為什麼說必至呢？其實小樽有什麼好吃的好看的，大概行前拜請估狗大神都查的差不多了

堺町沿路上當然有很多好吃的糕餅點心，可是唯一在還沒出發前，就讓我在台灣對著電腦流口水的

就是北菓樓的妖精之森年輪蛋糕！！所以看到北菓樓就馬上衝進去了 XDDDDDDDDDDDDD

基於背的東西實在已經很多，而且新千歲機場裡面什麼都有得買，所以我只是打算進去試吃一下

然後呢..............我就一直試吃.........................>/////////////////////< (所謂一直大概就是我吃了快十次......)

還好日本店家真的都很有禮貌，也沒人白眼我，就讓我當一次沒水準的觀光客吧 Orz

LeTAO與銀之鐘咖啡就沒那麼重要，所以LeTAO我們只是經過，連門都沒進去

不過他們的綠色大門真的是蠻有特色的，很搶眼，可惜遊客太多，一直抓不到背景沒遊客的畫面

銀之鐘的賣點是喝完咖啡可以把咖啡杯洗一洗帶回家，不過咖啡實在不怎麼樣，昨天在北工房喝的咖啡好喝多了 = =

在銀之鐘喝咖啡的同時，外面的天空也開始飄起雨來，而且有越下越大的趨勢

我們決定喝完後啟程回到小樽運河著名的倉庫區附近，等待日落後運河旁的燈點起來

回到小樽運河倉庫區的途中，我們經過色內十字路

色內十字路也是情書的點之一，在這個有小樽郵便局的十字路口，長大後的女藤井樹與博子在此碰面

其實也不算是碰面，博子即將離開小樽，在這個十字路口正好與騎著腳踏車的女藤井樹擦身而過

兩個容貌一模一樣的人，在這個十字路口有了短暫的重疊

我們到了這個十字路口，開始研究樹是從哪個方向騎腳踏車過來，博子當時是站在哪裡

包括考慮到樹應該是從山上的家下來，電影中路燈的樣子等等因素，但結果還是 ---------------- 研究不出來............Orz

所以我們在這裡留下了幾張影像，隨著雨越來越大，我們趕忙前往運河倉庫區

並且在運河倉庫區的一家德式酒吧休息 (兩個人又都睡死...)，等待運河燈亮起，北海道第三天的行程，在大雨中劃下句點.....

回到札幌後，我們在札幌站地下街找了一家拉麵，當作晚餐

原本是要找札幌著名的湯咖哩，可是已經很晚了，我們也不曉得湯咖哩在哪裡

只好看到拉麵又走進去，免得拖到更晚沒東西吃

這次吃個是地下街的「華龍」拉麵，可能是昨天被青葉拉麵震撼到，這家感覺就還好.....

有趣的是在吃的同時，陸續有好幾個穿著西裝，看起來就像是在某某商社工作的年輕人

可能是下班了，回家前也進來吃碗拉麵

那種感覺好像走進日劇的場景一樣，真實的呈現在眼前

也許我們真的受日本文化影響太深，到了日本，總是有種走進電視畫面裡的感覺吧 : )

小樽街道的異國風情，巧妙的與日本人給人的感覺相融合，傍晚的雨更讓運河透出一種朦朧的美，這是一個充滿思念的城市 : )
